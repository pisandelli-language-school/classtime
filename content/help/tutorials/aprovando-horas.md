# Como Aprovar Horas

Se você é um Gerente, sua tarefa semanal é revisar e aprovar os timesheets dos professores.

## O Processo de Aprovação

1.  Vá para o menu **Aprovações** no topo da tela.
2.  Localize o Professor na lista.
    *   Use o filtro de "Semana" se precisar ver períodos anteriores.
3.  Clique em **Ver Detalhes** (ícone de olho ou seta).
4.  Revise os lançamentos:
    *   Verifique se o total de horas bate com o esperado.
    *   Veja se há observações de faltas.
5.  **Tomar Decisão**:
    *   **Aprovar**: O status muda para APROVADO e libera para pagamento.
    *   **Rejeitar**: O status volta para REJEITADO. O professor será notificado (necessário informar motivo).

::alert{type="warning"}
Uma vez aprovada, a semana é travada para edição pelo professor. Apenas um admin Root pode reabri-la.
::
